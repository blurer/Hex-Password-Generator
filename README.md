Hex password generator [a-f,A-F,0-9]

# Installation
## Requirements
- Python 3

## Usage
```
git clone link
cd to directory
chmod +x pwgen.py
./pwgen.py
```
You'll be asked:
- How many characters do you want the password to be?
- How many passwords do you want?
